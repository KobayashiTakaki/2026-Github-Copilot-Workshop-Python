## フロントエンドの状態変数

タイマーの状態管理はすべてクライアントサイドのJavaScript（`timer.js`）で行われています。

| 変数名 | 型 | 説明 |
|--------|----|------|
| `WORK_MINUTES` | `number`（定数） | 作業セッションの分数（25分） |
| `BREAK_MINUTES` | `number`（定数） | 休憩セッションの分数（5分） |
| `timer` | `number \| null` | `setInterval` のタイマーID |
| `remainingSeconds` | `number` | 残り秒数（初期値：1500秒） |
| `isRunning` | `boolean` | タイマー動作中フラグ |
| `isWork` | `boolean` | 作業中フラグ（falseなら休憩中） |

## 進捗表示（ハードコード）

現在、画面上の進捗データは `index.html` にハードコードされた静的な値です。

```html
<span class="progress-count">4</span>      <!-- 完了数 -->
<span class="progress-total">1時間40分</span> <!-- 集中時間 -->
```

サーバーとのデータ通信は実装されておらず、ページ更新のたびにこの固定値が表示されます。
