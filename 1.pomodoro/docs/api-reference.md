# API リファレンス

ポモドーロタイマーアプリのAPIエンドポイント一覧です。

---

## エンドポイント一覧

### GET /

メイン画面のHTMLを返します。

**リクエスト**

```
GET /
```

**レスポンス**

- ステータスコード: `200 OK`
- Content-Type: `text/html`
- ボディ: `templates/index.html` をレンダリングしたHTMLページ

**例**

```
GET http://localhost:5000/
```

レスポンスとして、ポモドーロタイマーのUI（HTML）が返されます。

---

## 備考

現在のバックエンド実装は `/` エンドポイントのみです。進捗データの保存・取得API（`/progress` など）は設計上の計画段階にあり、現時点では実装されていません。

フロントエンドのタイマーロジックはすべてクライアントサイド（JavaScript）で動作します。
